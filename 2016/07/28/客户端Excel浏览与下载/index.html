      客户端Excel浏览与下载

      Post on 2016-07-28

      Excel文件的上传与下载在网站上使用的场景并不多，主要是在一些web管理后台上比较常使用到，在这样的网站上传Excel文件方便的的批量上传数据，前提是把所要的数据以一定的表格格式存在Excel文件；而下载Excel文件可以把想要的的数据保存在在Excel表格中方便统计与查看。

      在这篇文章中将要介绍如何通过浏览器JavaScript在浏览器浏览Excel表格，又如何下载网页上原有的table表格，并且今天还要给IE正名，IE浏览器也不是一无是处的。

      上传部分Excel只能使用ActiveXObject(“Excel.application”)对象在浏览器中显示Excel表格，而在非IE浏览器中因为不支持ActiveXObject主件对象，所以也不能在浏览器中预览Excel表格，现在IE终于有一方面比其他非IE浏览器强的地方，为了更好的兼容性，在多内核的浏览器中设置默认使用内核为IE内核，设置代码在上面出现过<meta name="renderer" content="ie-stand">,在这里设置的多内核浏览器为IE标准模式，这种模式下选择内核是IE9以上的IE内核，除了标准模式还可以通过上面方法选择兼容模式，浏览器的兼容模式是使用IE7内核作为默认内核，设置代码为<meta name="renderer" content="ie-comp">。下面就来看看IE是如何浏览Excel表格：

       //IE上传
          function read_excel(path){
              var filePath=path,sheet=””,temp_Str=””;
              try{
                  var oXL = new ActiveXObject(“Excel.application”); //创建Excel.Application对象
              }catch(err){
                  alert(err);
              }
              var oWB = oXL.Workbooks.open(filePath);
              //循环输出sheet
              for(var sheet_id=1;sheet_id<=oXL.Sheets.Count;sheet_id++){
                  var temp_Str=””;
                  oWB.worksheets(sheet_id).select();
                  var oSheet = oWB.ActiveSheet;
                  var rowcount=oXL.Worksheets(sheet_id).UsedRange.Cells.Rows.Count ;
                  var colcount=oXL.Worksheets(sheet_id).UsedRange.Cells.Columns.Count;
                  //循环Excel表中行
                  for(var i=1;i<=rowcount;i++){
                      var temp_Row=””;
                      //循环Excel表行中列
                      for(var j=1;j<=colcount;j++){
                          var temp_data=””;
                          if (typeof(oSheet.Cells(i,j).value)==’date’){ //单元格内容是日期格式时的读取问题
                              d= new Date(oSheet.Cells(i,j).value);
                              temp_data=d.getFullYear()+”-“+(d.getMonth() + 1)+”-“+d.getDate();
                          }else{
                              temp_data=$.trim(oSheet.Cells(i,j).value);
                          }

                      temp_Row+="<td>"+temp_data+"</td>";
            }
            temp_Str+="<tr>"+temp_Row+"</tr>";
        }
        sheet+="<caption align='top'>"+oXL.Worksheets(sheet_id).Name+"</caption><table id='"+oXL.Worksheets(sheet_id).Name+"' border=1 cellpadding=10>"+ temp_Str+"</table><a href='javascript:void(0);' class='download' download>"+oXL.Worksheets(sheet_id).Name+"</a>"; 
    }
    return sheet;
    oXL.Quit();
    CollectGarbage();//清除内存
}</pre>

      上面就是IE内核下的Excel表格浏览代码，下面带来的是在浏览器上下载table表格：

      //IE下载，通过打开一个窗口在新的窗口保存Excel
          function saveCode(obj,name){
              var winname = window.open(‘’, ‘_blank’, ‘top=10000’);
              var strHTML = document.all(obj).outerHTML;
              winname.document.open(‘text/html’, ‘replace’);
              winname.document.writeln(strHTML);
              winname.document.execCommand(‘saveas’,’’,name+’.xls’);
              winname.close(); //关闭
          }
          //IE下载，通过ActiveXObject(“Excel.Application”)打开Excel并把table复制到Excel中在手工保存
          function toExcel(id){
              window.clipboardData.setData(“Text”,document.all(id).outerHTML);
              try
              {
                  var oXL = new ActiveXObject(“Excel.Application”);
                  var ExWBk = oXL.workbooks.add();
                  var ExWSh = ExWBk.worksheets(1);
                  oXL.DisplayAlerts = false;
                  oXL.visible = true;
              }
              catch(e){
                  alert(e.message);
                  return false;
              }
              ExWBk.worksheets(1).Paste;
          }

      这些方式我也是在网上找到的，前面的IE浏览Excel，以及后面IE保存Excel的两种方法都比较好理解，但最后一个使用非IE的保存方法较为抽象，以我的理解就是先把表格数据嵌入template的格式，再把这种格式的字符串通过encodeURIComponent()编码unescape()解码和.btoa()转化为base64数据，然后通过类似于HTML5下载的形式把表啊个保存下来。

      最后来看一看这些方法是如何使用的：
