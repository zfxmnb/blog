      javascript原型

      Post on 2016-07-03

      在我的理解中javascript的原型就是对象object的原始属性，而原型链就是object的属性合集（我是这么理解的），那么如何设置对象的原型呢。

      首先一般在javascript中的对象都是由类实例化而来的，我们最常见的就是实例化function类，在这里也就以此为例来说明如何来给对象加原型。

          f=function(user,pwd){
              this.user=user;
              this.pwd=pwd;
          };
          f.prototype.fn1=function(username,password){
              console.log(“用户名：”+this.user+”，密码：”+this.pwd);
          };
           f.prototype.fn2=function(username,password){
              console.log(“用户名：”+username+”，密码：”+password);
          };
          o=new f(“username1”,”123456”);//用户名：username1，密码：123456
          o.fn1();
          o.fn2(“username2”,”654321”);//用户名：username2，密码：654321

      在这里我首先是创建了一个function类，在通过prototype给类添加原型，这里添加了两个原型用于对比，然后使用new f类来得到o对象，同时给o对象添加了user和pwd属性，再然后就是执行o对象的fn1方法和fn2方法，fn1方法是输出对象属性user和pwd，fn2是是通过传入值然后输出的，有没有发现这和jquery很像，是的在jquery中使用了大量的原型。不信可以通过遍历的方法把jquery中的对象属性方法遍历出来。

      for(var key in o){
              console.log(key+”:”+o[key])//属性或方法名:属性方法
          }

      以上就可以把对象的说有属性方法遍历出来了，这种方法不仅可以遍历出js中的对象还可以把元素对象中的属性方法遍历出来。

      var img=document.getElementById('img');
for(var key in img){
    console.log(key+":"+img[key]+";")
}
